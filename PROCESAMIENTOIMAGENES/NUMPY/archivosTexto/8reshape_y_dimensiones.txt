Reshape y Dimensiones en NumPy
------------------------------

Cambiar la forma de un array:
- arr.reshape(new_shape): Cambia la forma del array
- arr.shape: Devuelve la forma actual
- arr.ndim: Devuelve el n√∫mero de dimensiones

Ejemplo:
```python
import numpy as np
arr = np.arange(12)
print(arr.shape)         # (12,)
arr2 = arr.reshape((3, 4))
print(arr2.shape)        # (3, 4)
print(arr2.ndim)         # 2
```
