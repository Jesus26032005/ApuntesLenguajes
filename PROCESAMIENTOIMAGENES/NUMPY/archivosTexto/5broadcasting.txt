Broadcasting en NumPy
---------------------

El broadcasting permite realizar operaciones entre arrays de diferentes formas y tamaños, siguiendo reglas específicas para expandir los arrays automáticamente.

Reglas de broadcasting:
- Si los arrays tienen diferente número de dimensiones, se añaden dimensiones de tamaño 1 a la izquierda.
- Los tamaños de cada dimensión deben ser iguales o uno de ellos debe ser 1.

Ejemplo:
```python
import numpy as np
A = np.array([1, 2, 3])
B = np.array([[10], [20], [30]])
print(A + B)
```
